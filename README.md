# SAD_project

Este Datamart foi desenvolvido para apoiar a análise epidemiológica,
o monitoramento de casos e a tomada de decisão em saúde pública,
utilizando dados do Sistema de Informação de Agravos de Notificação (SINAN).









Autor: Leonardo Silva

Tecnologias utilizadas:
• Pentaho Data Integration (ETL)
• PostgreSQL (Data Mart)
• Power BI (Visualização)

Fonte dos dados:
Sistema de Informação de Agravos de Notificação – SINAN HIV/AIDS
Ministério da Saúde – Brasil
